1. Introduction

This dashboard was built to make planning church events simpler. It helps you:

  • Plan events — couples meetings, Sunday celebrations, baptisms, Christmas programmes, and more.
  • Delegate tasks to the right team — media, music, ushering, kids, decoration, food, and any custom team you create.
  • Rotate volunteers fairly — the app suggests people who haven't served recently, so the same few aren't always doing everything.
  • Track the budget — planned vs actual spend, per event.
  • Share the plan instantly — copy a WhatsApp-ready summary, download an Excel sheet, or print a clean list.

Everything saves automatically to the cloud. When Pauline adds or changes anything, everyone viewing the link sees the update within a few seconds — no refresh needed.

Opening the dashboard

3. Overview tab — your home page

The Overview gives you the big picture at a glance. It shows four coloured cards across the top:

  • Upcoming Events — number of events dated today or later.
  • Tasks To Do — tasks across all events that aren't marked done yet.
  • Volunteers — total people in your roster.
  • Total Spend — actual spend across all events in euros.

Below the cards:

  • Next Event — the very next event on the calendar, with its task count and how many volunteer assignments have been made.
  • Recent Activity — the last ten changes made, so you always know what's been going on. Admin can click Clear activity to wipe the log when it gets long.

4. Events tab — creating and managing events

This is where you add, edit, and delete events.

Creating a new event

  1. Make sure you are signed in as admin.
  2. Click the + New Event button.
  3. Fill in:
    • Name — e.g. Christmas Celebration 2026 or Couples Meeting — April.
    • Date — pick from the calendar.
    • Location — defaults to "Church". Change if it's somewhere else.
    • Description — a one-liner about the event (optional).
  4. Click Save.

Viewing and managing events

Each event shows as a card with its date, location, description, and task progress (3/10 tasks complete). On each card:

  • Open tasks — jumps to the Tasks & Teams tab for that event.
  • Edit — change event details.
  • Delete — removes the event and all its tasks and expenses. The app will ask you to confirm first.

5. Tasks & Teams tab — the main workspace

This is where most of the planning happens. You'll use it for every event.

Step 1 — Pick an event

Use the dropdown at the top right of the tab to select which event you're planning.

Step 2 — Add tasks

You have two ways to add tasks:

  1. From the standard list (recommended for most events):
    • Click Add from standard list…
    • A checklist appears with common church-event tasks: decoration, food, media, ushering, kids, cleanup, and more.
    • Tick only the ones you need. You can click Clear all first, then tick selectively.
    • Click Add selected. The tasks appear on the board instantly.
  2. Add a custom task:
    • Click + Add Task.
    • Enter the title, pick a category, optionally add estimated hours and notes.
    • Click Save.

Step 3 — Assign volunteers to each task

Every task appears as a card. On each card:

  • The task category is shown as a coloured tag (e.g. Media, Food, Decoration).
  • A Rotation suggestion shows up to three people who haven't served recently. Click Apply to assign the top suggestion in one click.
  • The Assign… dropdown lets you pick anyone from the roster. For media/music/ushering/kids tasks, members of that team appear first, with others grouped below.
  • Each person's name shows how many tasks they already have in this event (e.g. "Karen · 2 tasks"). This helps avoid overloading.
  • Click the × next to an assigned name to remove them.
  • Click Mark done when the task is completed during the event.

Step 4 — Auto-fill rotation (optional but handy)

Click Auto-fill rotation at the top of the tab. The app will go through every task that has no volunteer yet and assign the best rotation pick — while making sure no one gets more than 3 tasks for this event. Pauline can always adjust afterwards.

Step 5 — Share the plan

A Summary card appears at the top once tasks are added. It shows:

  • Total tasks, how many are assigned, how many still need a volunteer, how many are done.
  • All tasks grouped by category, with the volunteer names under each.

Three buttons:

  • Copy for WhatsApp — copies a ready-to-paste message for the church WhatsApp group, with the event name, date, tasks, and assigned names grouped neatly.
  • Download Excel — saves an .xlsx file with three sheets: Tasks, People (per-person task count), and Budget.
  • Print — opens a printable version of the summary in a new tab.

6. People tab — teams and volunteers

Pre-configured teams

On first use, the dashboard has five default teams: Leadership, Media, Music, Ushering, Kids. Pastor and Pauline are automatically part of every team.

Loading the NLC roster in one click

Click Load NLC Team to add everyone from the pre-built NLC roster (Kingson, Karen, Naveen, Tabitha, and so on). People already in the list are skipped automatically, so it's safe to click more than once.

Adding someone yourself

  1. Click + Add Person.
  2. Enter the name, tick the teams they belong to (a person can be in more than one), add a note if helpful.
  3. Click Save.

Adding or removing a team

  • + Add Team — create a new team like Prayer, Youth, or Kitchen.
  • Click the × next to a custom team name in the filter bar to delete it (default teams can't be deleted).

Filtering by team

The row of team chips (All, Leadership, Media, …) at the top of the People tab filters the list. Click a chip to see only that team. Each chip shows the number of people currently in it.

Load indicator

Each person's card shows a small coloured number — how many tasks they have in the currently selected event. Green means light load, yellow means moderate, red means they're busy. A simple way to spot who's being asked to do too much.

7. Budget tab — planned vs actual spend (admin-only)

The Budget tab is visible only when signed in as admin, because it contains financial information.

  1. Pick an event from the dropdown at the top right.
  2. Three totals show across the top: Planned, Actual, Remaining.
  3. Click + Add Expense to record a cost:
    • Item — what was bought (e.g. "Cake", "Decoration supplies").
    • Category — Decoration, Food, Shopping, Hospitality, Kids, Media, or Other.
    • Paid by — who paid (for reimbursement tracking).
    • Planned and Actual amounts in euros.
  4. Edit or delete any row at any time using the buttons on the right.
  5. Click Download Excel right here on the Budget tab for a full report of the selected event (Tasks, People, and Budget sheets).

The same Excel report is also available from the Tasks tab's Summary card.

8. A typical event — step-by-step workflow

Planning a Christmas Celebration, start to finish:
  1. Sign in as Admin.
  2. Events tab → + New Event → enter "Christmas Celebration 2026", date 25 December 2026, location Church.
  3. Tasks & Teams tab → select the new event from the dropdown.
  4. Click Add from standard list… → tick the tasks relevant (venue decoration, meal preparation, worship, photography, kids ministry, cleanup, and so on).
  5. Click Auto-fill rotation → the app assigns one volunteer per task, fairly rotated.
  6. Review each task card. Adjust assignments manually where you want specific people (e.g. "Karen should lead worship this time").
  7. Budget tab → add planned expenses: decoration ₹X, food ₹Y, cake ₹Z.
  8. Click Copy for WhatsApp → paste into the church group chat to let everyone see their roles.
  9. On the day, open the dashboard on a phone → tick Mark done on each task as it's completed.
  10. After the event, go back to Budget and fill in Actual amounts for each expense. Done.

9. Tips & common questions

Can more than one person edit at the same time?

Yes. Pauline can be on her laptop while another admin helper is on a phone — all changes sync live across everyone viewing the dashboard, usually within 1–2 seconds.

Does it work on a phone?

Yes — the layout adjusts to small screens. Great for updating task status during an event.

What if someone isn't in the list yet?

Sign in as Admin, go to People, click + Add Person. Once added, they can be assigned to tasks in any event, past or future.

Can I delete a mistake?

Yes — every event, task, person, and expense has a Delete button (visible when signed in as admin). The app asks to confirm first. Be aware that deleting an event removes all its tasks and expenses.

What if I want a new team (e.g. Prayer or Youth)?

On the People tab, click + Add Team and give it a name. The new team will appear everywhere — in the filter bar, in person profiles, and throughout the app.

What about the rotation suggestion — does it always pick the same person?

No. It looks at who was assigned in the most recent past event (those people drop down the list) and who has done the fewest tasks overall (they move up). For team-specific tasks like Media or Music, it also prefers people in that team. Pauline's manual choice always takes priority — the suggestion is just a helpful hint.

Is our data safe?

Yes — the data lives in Google's Firebase cloud. Only people with the dashboard link can view it, and only someone with the passcode can edit. The link is not listed publicly anywhere.

Is there help available inside the app?

Yes — there's a Help tab at the top of the dashboard itself. It contains the same instructions as this document and is updated automatically whenever a new feature is added. That way the help is always in sync with the latest version of the app.

Who do I ask for help?

Please reach out to Omprakash — he built this dashboard and is happy to add features, adjust teams, or help with anything that isn't working the way you expect.
